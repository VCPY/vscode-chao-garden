# Chao Garden
Add a few chao to your VS Code IDE with this plugin!

![example](https://github.com/VCPY/vscode-chao-garden/blob/main/Chao%20Garden%20Example.gif)

## Installation
You can add this extension through the VS Code marketplace or through the extension panel in your VS Code application.

## Configuration
Open the settings using `CTRL + ","` on Windows/Linux and `CMD + ","` on MacOs. You can find the settings for the chao garden when searching for "chao garden".

In the settings, you can choose the size of the chao. Choose between small, medium and large.

## Usage
A chao garden window will appear in your explorer tree. You can add a specific type of chao or add a random chao, and name them!

They will explore their new home immeditately. Have you tried hovering over them with your mouse?
